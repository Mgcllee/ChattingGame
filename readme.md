# Multi-User chatting program
---

1. 10 ~ 100만  클라이언트 접속 허용해보기
2. 유저가 접속하는 방을 따로 만들어 관리하기
3. 채팅 중 비속어 등을 처리할 로직 만들기 ('*' 기호로 변환 등)
4. 방 생성은 조건(100명 충족 or 생성된 시간)이 충족되면 채팅 시작

<br/>

## 현재 접속 상황

> netsh int ipv4 set dynamicport tcp start=1024 num=64511

위 명령어를 통해서 약 16'000 개의 연결만 가능한 상태에서 약 64500 개 까지 연결이 가능하게 됨.
